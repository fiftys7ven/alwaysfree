Get started

Before adding something, you need to:

  1. Fork this repository.
  2. Read our contributing rules and follow them.

Adding new service

  1. Go to categories folder.
  2. Find category that is appropriate for the service.
  3. Open & Edit file (APPROPRIATE CATEGORY).html
  4. Learn about HTML tables. You can also use sample code.
  5. Add Title & Description & URL (In Title).
  6. Done!

Adding new category

  1. Go to categories folder.
  2. Create a file named (YOUR CATEGORY NAME).html
  3. Open & Edit file (YOUR CATEGORY NAME).html
  4. Learn about HTML tables.
  5. Add this code to your category.
  6. Add Some Services.
  7. Done!
